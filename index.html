Project Overview

MoonAI models predator–prey dynamics using evolutionary algorithms and neural networks. The system evolves both agent attributes (speed, vision, stamina, reproduction rate) and behavioral strategies through the NEAT algorithm. The objective is to analyze how genetic representations influence adaptability in complex environments and to evaluate the performance of evolutionary computation techniques.

Technologies Used

  • C++ / SFML – Real-time simulation & visualization
  • CUDA – GPU-accelerated evolutionary computation
  • Python (Pandas, Matplotlib) – Data analysis & visualization
