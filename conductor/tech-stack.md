# Technology Stack

## Frontend
- **HTML5:** Semantic markup for project content and document links.
- **CSS3:** Custom styles defined in `styles.css` using a grid-based layout.
- **Google Fonts:** "Source Sans 3" for typography.

## Deployment & Hosting
- **GitHub Pages:** Static site hosting directly from the repository.
- **GitHub Actions:** Automated deployment workflow (defined in `.github/workflows/static.yml`).

## Development Tools
- **Gemini CLI:** For project management and autonomous implementation using the Conductor methodology.
- **Git:** Version control for project files and documentation history.
